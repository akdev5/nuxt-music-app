# Nuxt Music Manager

## Technology Used

* Vue
* Nuxt.js
* Node.js
* Express.js

## Project setup
On both of server and client side,

```
npm install
```

### Compiles and hot-reloads for development on Client
```
cd client
npm run start
```

### Run server for development on Client
```
cd server
node index.js
```

## Scripts
- `npm test` - run client and server tests once
